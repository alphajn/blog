深度图 ​

Demo ​

Install ​

bash
# 使用 npm
npm i @fe/depth

# 使用 yarn
yarn add @fe/depth

# 使用 pnpm
pnpm add @fe/depth

使用 ​

vue
<script setup lang="ts">
import { onMounted, reactive } from 'vue';
import { Depth } from '@fe/depth';

const state = reactive({
    el: null,
    chart: null,
    buys: [
        [200000.69, 1000.0004],
        [300000.72, 2000.0006],
        [400000.3, 60.999],
    ],
    sells: [
        [200001.6329, 1000.0005],
        [300001.72, 200.0006],
        [400001.3, 60.999],
    ],
});

onMounted(() => {
    state.chart = new Depth(state.el, {
        theme: 'light',
    });
    state.chart?.setData(state.buys, state.sells);
});
</script>

<template>
    <div :ref="(el) => state.el = el" />
</template>

实例函数 ​

setData() ​

设置图表数据。

  • 类型

    ts
    type ArrayList = (string | number)[]

function setData(buy: ArrayList[], sell: ArrayList[]): void;

  • 示例

    ts
    const buys = [
    [200000.69, 1000.0004],
    [300000.72, 2000.0006],
    [400000.3, 60.999],
];
const sells = [
    [200001.6329, 1000.0005],
    [300001.72, 200.0006],
    [400001.3, 60.999],
];
setData(buys, sells);

formatList() ​

  • 类型
