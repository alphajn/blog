基础版vue组件 ​

基于基础版开发, 自适应移动端。

注意

只支持vue3版本。

Demo ​

Install ​

bash
# 使用 npm
npm i @fe/kline @fe/number klinecharts

# 使用 yarn
yarn add @fe/kline @fe/number klinecharts

# 使用 pnpm
pnpm add @fe/kline @fe/number klinecharts

使用 ​

vue
<script setup lang="ts">
import { SimpleChart } from '@fe/kline/components';

const state = reactive({
    loading: false,
    theme: 'dark',
    lang: 'en-US',
    list: [],
    latest_data: null,
});

const methods = {
    loadMore(ts: number | null, cb: (list) => void) {
        // ::TODO
    },
}
</script>

<template>
    <SimpleChart
        :loading="state.loading"
        :theme="state.theme"
        :lang="state.lang"
        :list="state.list"
        :latest-data="state.latest_data"
        :load-more="methods.loadMore"
        watermark="Kline"
    />
</template>

API ​

属性 ​

属性名说明类型默认值
loading数据获取中展示loading样式booleanfalse
theme主题string''
lang语言string''
showMa默认展示ma指标booleantrue
showVol默认展示成交量指标booleantrue
limitSize分页条数，判断是否还有更多number300
watermark水印HTMLElement | string''
list初始列表Array<KlineData>[]
latestData最新数据KlineDatanull
loadMore加载历史数据回调(ts: number | null, cb: (list: KLineData[]) => void): void() => void

Theme类型 ​

ts
type Theme = 'en-US' | 'zh-CN' | 'zh-TW' | 'ja-JP' | 'vi-VN' | 'es-ES' | 'ru-RU' | 'tr-TR' | 'pt-BR' | 'fr-FR' | 'th-TH' | 'id-ID';

KlineData类型 ​

ts
type KlineData = {
    timestamp: number; // 时间戳，毫秒级别，必要字段
    open: number; // 开盘价，必要字段
    high: number; // 最高价，必要字段
    low: number; // 最低价，必要字段
    close: number; // 收盘价，必要字段
    volume: number; // 成交量，非必须字段
}

默认导出 ​

名称说明类型
chart图表实例Chart

Chart类型 ​

ts
type Chart = KlineChart & {
    $updateTooltip: () => void // 更新tooltip
    $setTheme: (theme: `${Theme}`) => void // 更新主题
    $addDefaultMa: () => void // 手动添加ma指标
    $addVol: () => void // 手动添加成交量
} | null;
