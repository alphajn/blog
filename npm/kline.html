带菜单合并版本 ​

基于 Tradingview 26.003 开发，官方文档。

Demo ​

Install ​

bash
# 如果需要使用简版kline需要下载以下依赖
@fe/number klinecharts

# 使用 npm
npm i @fe/kline

# 使用 yarn
yarn add @fe/kline

# 使用 pnpm
pnpm add @fe/kline

使用 ​

注意

  1. 如果使用 TradingView 需要把对应版本的 charting_library 放到静态文件目录下，并设置 libraryPath 属性为 charting_library 路径
  2. 如果需要自定义样式需要把对应 css 文件放到 charting_library 目录下，并设置 custom_css_url 属性为文件名
vue
<script setup lang="ts">
import { Kline } from '@fe/kline/vue';
</script>
<template>

<Kline
    class="h-400px"
    :loading="loading"
    :theme="theme"
    :lang="lang"
    watermark="水印"
    :type="type"
    library-path="/public/charting_library/"
    :price-precision="pricePrecision"
    :quantity-precision="quantityPrecision"
    :symbol="symbol"
    :list="state.list"
    :latest-data="state.latest_data"
    :load-more="methods.loadMore"
/>
</template>
