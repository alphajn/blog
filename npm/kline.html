注意

只支持vue3版本。

支持同时使用 和 支持单独使用(基础版、专业版)

Install ​

bash
# 如果需要使用简版kline需要下载以下依赖
fejn-number klinecharts

# 使用 npm
npm i fejn-kline

# 使用 yarn
yarn add fejn-kline

# 使用 pnpm
pnpm add fejn-kline

使用 ​

注意

  1. 如果使用 TradingView 需要把对应版本的 charting_library 放到静态文件目录下，并设置 libraryPath 属性为 charting_library 路径
  2. 如果需要自定义样式需要把对应 css 文件放到 charting_library 目录下，并设置 custom_css_url 属性为文件名

ts
/** kline数据类型 */
export type KLineData = {
    timestamp: number // 简版kline时间
    open: number // 开盘价
    high: number // 最高价
    low: number // 最低价
    close: number // 收盘价
    volume?: number // 成交量
    turnover?: number // 成交额
};

/** tradingView数据类型 */
export type TvKlineData = Omit<KLineData, 'turnover' | 'timestamp'> & {
    time: number // 时间
    id: string // 唯一标识 `${symbol}_${interval}`
};
