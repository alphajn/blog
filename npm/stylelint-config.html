@fe/stylelint-config ​

样式格式化

stylelint文档

Install ​

bash
pnpm add -D stylelint @fe/stylelint-config

配置 .stylelintrc.js ​

js
module.exports = {
    extends: '@fe/stylelint-config',
};

添加命令到 package.json ​

json
{
  "scripts": {
    "lint:style": "stylelint \"**/*.{scss,css,vue}\" --fix"
  }
}

配置 VS Code 自动修复 ​

json
{
  "editor.codeActionsOnSave": {
    "source.fixAll.stylelint": true
  },
  "stylelint.validate": [
    "css",
    "less",
    "postcss",
    "scss",
    "vue",
    "sass"
  ]
}
