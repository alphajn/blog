订单簿列表 ​

Demo ​

Install ​

bash
pnpm add @fe/vue

使用 ​

vue
<script setup lang="ts">
import { OrderBookTable } from '@fe/vue/components';
</script>

<template>
    <OrderBookTable
        :list="sells"
        side="sell"
        @click="clickHandle"
    />
</template>

API ​

属性 ​

属性名说明类型默认
side订单簿控制价格颜色和占比颜色'buy' | 'sell' | ''''
list展示列表array[]
buyColor买色值string#15bb7a
sellColor卖色值string#d2334d
lineHeight单元格行高string16px
newIndex高亮背景索引(小于newIndex会有闪烁效果)number0

list列表属性 ​

属性名说明类型默认
side控制价格颜色'buy' | 'sell' | ''''
price价格string''
quantity数量string''
column3第三列展示数据string''
scale控制背景宽度，需要小于1string''

事件 ​

事件名说明Type
click点击每一条时触发(price: string) => void
