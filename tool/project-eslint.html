配置 eslint ​

  1. 使用封装扩展 @fe/eslint-config-vue
  2. 文档 @antfu/eslint-config

安装 ​

bash
pnpm add -D eslint @antfu/eslint-config eslint-config-airbnb-base

配置 .eslintrc.js ​

js
module.exports = {
    root: true,
    extends: [
        '@antfu/eslint-config',
        'airbnb-base',
    ],
};

添加命令到 package.json ​

json
{
  "scripts": {
    "lint": "eslint .",
    "lint:fix": "eslint . --fix"
  }
}

配置 VS Code 自动修复 ​

json
{
  "editor.formatOnSave": false,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "eslint.validate": [
    "javascript",
    "typescript",
    "javascriptreact",
    "typescriptreact",
    "vue",
    "html",
    "json",
    "jsonc",
    "json5",
    "markdown",
    "yaml"
  ]
}
